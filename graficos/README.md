# Análisis comparativo del tiempo de ejecución de Bubble Sort, Selection Sort y Quick Sort

## Descripción

Este proyecto presenta un análisis comparativo del tiempo de ejecución de los algoritmos de ordenamiento Bubble Sort, Selection Sort y Quick Sort mediante experimentos realizados en Python 3.13.

El estudio evalúa el comportamiento de los algoritmos utilizando tres escenarios de prueba:

- Datos aleatorios
- Datos ordenados
- Datos en orden inverso (peor escenario)

Además, el programa calcula estadísticas como promedio, mediana y desviación estándar de los tiempos de ejecución, generando automáticamente tablas y gráficos para el análisis de resultados.

---

## Objetivo

Comparar experimentalmente el rendimiento de Bubble Sort, Selection Sort y Quick Sort mediante la medición de sus tiempos de ejecución bajo diferentes escenarios y tamaños de entrada.
