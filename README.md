# Análisis comparativo del tiempo de ejecución de Bubble Sort, Selection Sort y Quick Sort

## Descripción

Este proyecto presenta un análisis comparativo del tiempo de ejecución de los algoritmos de ordenamiento Bubble Sort, Selection Sort y Quick Sort mediante experimentos desarrollados en Python 3.13.

El estudio evalúa el comportamiento de los algoritmos en tres escenarios de prueba:

- Datos aleatorios
- Datos ordenados
- Datos en orden inverso (peor escenario)

Además, el programa calcula automáticamente el tiempo promedio, la mediana y la desviación estándar de los tiempos de ejecución, generando tablas y gráficos utilizados en el artículo científico.

---

## Objetivo

Comparar experimentalmente el rendimiento de los algoritmos Bubble Sort, Selection Sort y Quick Sort mediante la medición de sus tiempos de ejecución bajo diferentes escenarios y tamaños de entrada.
